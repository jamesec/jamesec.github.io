# Best way to save your chat history with ChatGPT
`January 22, 2025`

I use ChatGPT in a minimalist, low-key way, without logging into the system. That is, I simply [start a new temporary chat](https://chatgpt.com/?temporary-chat=true) each time.

> Temporary Chat  
> This chat won't appear in history or be used to train our models. For safety purposes, we may keep a copy of this chat for up to 30 days.

However, I’d like to save my conversations for future reference. Previously, I used a browser screenshot extension, namely `Awesome Screen Recorder & Screenshot`, to do so.

I’ve now discovered a better solution, which, as I understand it, is the best way to save/export chat history: using the browser extension `SingleFileZ`.

You can install the browser extension for Chromium/Chrome [here](https://chromewebstore.google.com/detail/singlefilez/offkdfbbigofcgdokjemgjpdockaafjg?hl=en). If you're using a different browser, just search for `SingleFileZ name_of_your_browser`.

You can save the webpage with ChatGPT as an .mhtml file by using "Save Page As..." in Chromium/Chrome and selecting "Webpage, SingleFile" as the format. However, the resulting .mhtml file will be significantly larger than the file generated by SingleFileZ.
